# MSFvenom Payload Creator

## Requirements

The Metasploit framework.

## Installation

Is preinstalled in Kali. On other *nix, to install:

    git clone https://github.com/g0tmi1k/mpc

Set execute permission on the `msfpc.sh` file:

    cd mpc/
    chmod +x msfpc.sh

Start her up with:

    ./msfpc.sh
